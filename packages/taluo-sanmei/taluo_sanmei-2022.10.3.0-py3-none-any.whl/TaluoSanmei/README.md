# 塔罗三昧

## 下载

### Docker

```
docker pull apachecn0/taluo-sanmei
docker run -tid -p <port>:80 apachecn0/taluo-sanmei
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install taluo-sanmei
taluo-sanmei <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g taluo-sanmei
taluo-sanmei <port>
# 访问 http://localhost:{port} 查看文档
```
