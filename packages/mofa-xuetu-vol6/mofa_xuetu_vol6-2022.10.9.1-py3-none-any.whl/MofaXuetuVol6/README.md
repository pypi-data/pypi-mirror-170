# 魔法学徒06

## 下载

### Docker

```
docker pull apachecn0/mofa-xuetu-vol6
docker run -tid -p <port>:80 apachecn0/mofa-xuetu-vol6
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install mofa-xuetu-vol6
mofa-xuetu-vol6 <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g mofa-xuetu-vol6
mofa-xuetu-vol6 <port>
# 访问 http://localhost:{port} 查看文档
```
