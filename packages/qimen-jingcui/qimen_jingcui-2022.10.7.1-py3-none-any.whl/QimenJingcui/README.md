# 奇门精粹

## 下载

### Docker

```
docker pull apachecn0/qimen-jingcui
docker run -tid -p <port>:80 apachecn0/qimen-jingcui
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install qimen-jingcui
qimen-jingcui <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g qimen-jingcui
qimen-jingcui <port>
# 访问 http://localhost:{port} 查看文档
```
