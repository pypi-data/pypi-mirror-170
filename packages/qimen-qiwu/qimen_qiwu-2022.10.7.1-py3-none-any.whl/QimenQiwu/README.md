# 奇门启悟

## 下载

### Docker

```
docker pull apachecn0/qimen-qiwu
docker run -tid -p <port>:80 apachecn0/qimen-qiwu
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install qimen-qiwu
qimen-qiwu <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g qimen-qiwu
qimen-qiwu <port>
# 访问 http://localhost:{port} 查看文档
```
