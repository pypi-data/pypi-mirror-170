# 梅花新易

## 下载

### Docker

```
docker pull apachecn0/meihua-xinyi
docker run -tid -p <port>:80 apachecn0/meihua-xinyi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install meihua-xinyi
meihua-xinyi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g meihua-xinyi
meihua-xinyi <port>
# 访问 http://localhost:{port} 查看文档
```
