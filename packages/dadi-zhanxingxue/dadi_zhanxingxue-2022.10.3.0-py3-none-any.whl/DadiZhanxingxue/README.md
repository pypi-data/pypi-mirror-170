# 大地占星学

## 下载

### Docker

```
docker pull apachecn0/dadi-zhanxingxue
docker run -tid -p <port>:80 apachecn0/dadi-zhanxingxue
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install dadi-zhanxingxue
dadi-zhanxingxue <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dadi-zhanxingxue
dadi-zhanxingxue <port>
# 访问 http://localhost:{port} 查看文档
```
