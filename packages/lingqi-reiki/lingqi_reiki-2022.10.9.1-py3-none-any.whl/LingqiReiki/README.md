# 灵气REIKI

## 下载

### Docker

```
docker pull apachecn0/lingqi-reiki
docker run -tid -p <port>:80 apachecn0/lingqi-reiki
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install lingqi-reiki
lingqi-reiki <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g lingqi-reiki
lingqi-reiki <port>
# 访问 http://localhost:{port} 查看文档
```
