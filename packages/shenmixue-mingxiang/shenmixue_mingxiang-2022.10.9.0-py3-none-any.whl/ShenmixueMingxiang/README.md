```

docker run -tid -p <port>:80 apachecn0/shenmixue-mingxiang
# 访问 http://localhost:{port} 查看文档
```

```
pip install shenmixue-mingxiang
shenmixue-mingxiang <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
shenmixue-mingxiang <port>
# 访问 http://localhost:{port} 查看文档
```
