# 精油图鉴

## 下载

### Docker

```
docker pull apachecn0/jingyou-tujian
docker run -tid -p <port>:80 apachecn0/jingyou-tujian
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install jingyou-tujian
jingyou-tujian <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g jingyou-tujian
jingyou-tujian <port>
# 访问 http://localhost:{port} 查看文档
```
