# 赛斯书：灵界的讯息

## 下载

### Docker

```
docker pull apachecn0/lingjiede-xunxi
docker run -tid -p <port>:80 apachecn0/lingjiede-xunxi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install lingjiede-xunxi
lingjiede-xunxi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g lingjiede-xunxi
lingjiede-xunxi <port>
# 访问 http://localhost:{port} 查看文档
```
