# 活用水晶

## 下载

### Docker

```
docker pull apachecn0/huoyong-shuijing
docker run -tid -p <port>:80 apachecn0/huoyong-shuijing
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install huoyong-shuijing
huoyong-shuijing <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g huoyong-shuijing
huoyong-shuijing <port>
# 访问 http://localhost:{port} 查看文档
```
