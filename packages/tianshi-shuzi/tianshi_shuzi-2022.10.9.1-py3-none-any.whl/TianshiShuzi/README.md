# 天使数字

## 下载

### Docker

```
docker pull apachecn0/tianshi-shuzi
docker run -tid -p <port>:80 apachecn0/tianshi-shuzi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install tianshi-shuzi
tianshi-shuzi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g tianshi-shuzi
tianshi-shuzi <port>
# 访问 http://localhost:{port} 查看文档
```
