# 奇门秘占

## 下载

### Docker

```
docker pull apachecn0/qimen-mizhan
docker run -tid -p <port>:80 apachecn0/qimen-mizhan
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install qimen-mizhan
qimen-mizhan <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g qimen-mizhan
qimen-mizhan <port>
# 访问 http://localhost:{port} 查看文档
```
