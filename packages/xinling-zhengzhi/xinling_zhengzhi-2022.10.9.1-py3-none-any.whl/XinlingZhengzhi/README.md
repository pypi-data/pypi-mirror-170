# 赛斯书：心灵政治

## 下载

### Docker

```
docker pull apachecn0/xinling-zhengzhi
docker run -tid -p <port>:80 apachecn0/xinling-zhengzhi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install xinling-zhengzhi
xinling-zhengzhi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g xinling-zhengzhi
xinling-zhengzhi <port>
# 访问 http://localhost:{port} 查看文档
```
