# 尤尼希皮里

## 下载

### Docker

```
docker pull apachecn0/younixipili
docker run -tid -p <port>:80 apachecn0/younixipili
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install younixipili
younixipili <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g younixipili
younixipili <port>
# 访问 http://localhost:{port} 查看文档
```
