# 经典塔罗

## 下载

### Docker

```
docker pull apachecn0/jingdian-taluo
docker run -tid -p <port>:80 apachecn0/jingdian-taluo
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install jingdian-taluo
jingdian-taluo <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g jingdian-taluo
jingdian-taluo <port>
# 访问 http://localhost:{port} 查看文档
```
