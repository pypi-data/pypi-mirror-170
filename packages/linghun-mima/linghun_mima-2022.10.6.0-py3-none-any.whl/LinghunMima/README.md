# 灵魂密码

## 下载

### Docker

```
docker pull apachecn0/linghun-mima
docker run -tid -p <port>:80 apachecn0/linghun-mima
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install linghun-mima
linghun-mima <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g linghun-mima
linghun-mima <port>
# 访问 http://localhost:{port} 查看文档
```
