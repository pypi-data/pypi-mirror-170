# 富在工作

## 下载

### Docker

```
docker pull apachecn0/fuzai-gongzuo
docker run -tid -p <port>:80 apachecn0/fuzai-gongzuo
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install fuzai-gongzuo
fuzai-gongzuo <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g fuzai-gongzuo
fuzai-gongzuo <port>
# 访问 http://localhost:{port} 查看文档
```
