# 灵性之旅

## 下载

### Docker

```
docker pull apachecn0/lingxing-zhilv
docker run -tid -p <port>:80 apachecn0/lingxing-zhilv
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install lingxing-zhilv
lingxing-zhilv <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g lingxing-zhilv
lingxing-zhilv <port>
# 访问 http://localhost:{port} 查看文档
```
