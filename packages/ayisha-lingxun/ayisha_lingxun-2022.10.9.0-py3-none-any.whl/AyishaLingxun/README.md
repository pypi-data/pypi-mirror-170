# 阿乙莎灵讯

## 下载

### Docker

```
docker pull apachecn0/ayisha-lingxun
docker run -tid -p <port>:80 apachecn0/ayisha-lingxun
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install ayisha-lingxun
ayisha-lingxun <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g ayisha-lingxun
ayisha-lingxun <port>
# 访问 http://localhost:{port} 查看文档
```
