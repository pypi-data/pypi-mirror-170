# 灵气手札

## 下载

### Docker

```
docker pull apachecn0/lingqi-shouzha
docker run -tid -p <port>:80 apachecn0/lingqi-shouzha
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install lingqi-shouzha
lingqi-shouzha <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g lingqi-shouzha
lingqi-shouzha <port>
# 访问 http://localhost:{port} 查看文档
```
