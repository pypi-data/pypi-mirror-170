# 太乙考证

## 下载

### Docker

```
docker pull apachecn0/taiyi-kaozheng
docker run -tid -p <port>:80 apachecn0/taiyi-kaozheng
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install taiyi-kaozheng
taiyi-kaozheng <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g taiyi-kaozheng
taiyi-kaozheng <port>
# 访问 http://localhost:{port} 查看文档
```
