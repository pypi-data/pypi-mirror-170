# 秘密的秘密：实践幸福的吸引力法则

## 下载

### Docker

```
docker pull apachecn0/mimide-mimi
docker run -tid -p <port>:80 apachecn0/mimide-mimi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install mimide-mimi
mimide-mimi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g mimide-mimi
mimide-mimi <port>
# 访问 http://localhost:{port} 查看文档
```
